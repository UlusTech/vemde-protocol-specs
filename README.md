# Vemde Protocol Specification

**THIS IS CURRENTLY JUST AI, WE WILL RE-WRITE ALL!**

**Vemde** (Versioned Editing for MarkDown Environments) is an open protocol for real-time collaborative markdown editing. It defines how clients and servers communicate during a live editing session, leaving version control, transport, authentication, and persistence as implementation concerns.

Vemde is intentionally minimal. It is the session layer only — the equivalent of VSCode's Live Share, not a full editor or platform.
